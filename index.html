TheGeekPixel API Integration

This private backend application is designed to simplify product and order management for my own Etsy shop.

Features

  • Product Creation: Create and manage products in my backend, select where to publish them (Etsy, TikTok Shop, personal website), and synchronize data seamlessly.
  • Order Management: Receive Etsy orders in my backend, register them in the database, and request external APIs if needed for fulfillment.
